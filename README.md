# 🍝 Bella Vista - Authentic Italian Restaurant

This is a modern, responsive restaurant landing page built using HTML5 and CSS3.
It provides a clean and engaging user interface with sections for menu display, about, and reservation, making it ideal for showcasing a restaurant online.

------------------------------------------------------------------------

## ✨ Features

- 📱 **Responsive Design**  
  Fully optimized for mobile, tablet, and desktop using Flexbox and CSS Grid.

- 🎬 **Animated UI**  
  Smooth animations like `fadeInUp` and `slideInLeft` for better user experience.

- 🍽️ **Menu Section**  
  Displays categorized food items with pricing and hover effects.

- 📅 **Reservation Form**  
  Allows users to select date, time, and number of guests.

- 🎨 **Modern Styling**  
  Uses CSS variables for consistent colors, spacing, and shadows.
------------------------------------------------------------------------

## 🛠 Requirements
- Any modern web browser (Chrome, Edge, Firefox)
- No backend or server required

------------------------------------------------------------------------

## ⚙️ Installation
1. Clone the repository
git clone https://github.com/Shreyansh-Kumar-Singh/bella-vista.git

## ▶️ Usage
- Open the website in your browser
- Scroll through sections like Home, Menu, About, and Contact
- Fill out the reservation form (UI only)
